# Seller Images

Please add the following image files to this folder:

## Required Images:

1. **marketplace-icon.png** - Icon for MarketPlace link in navbar
2. **lazglobal-icon.png** - Icon for LazGlobal link in navbar
3. **philippines-flag.png** - Philippines flag icon for country selector
4. **lazada-app-icon.png** - Lazada App icon for login button
5. **google-icon.png** - Google icon for Google login button
6. **qr-code.png** - QR code for downloading seller app in footer

## Image Specifications:
- All icons should be PNG format with transparent background
- Navbar icons: 16x16px or 24x24px
- Flag icon: 20x20px or 24x24px
- App/Google icons: 24x24px or 32x32px
- QR code: 96x96px (w-24 h-24)
